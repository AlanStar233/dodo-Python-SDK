## 项目简介

​	本项目基于 [**DoDo开放平台-内测平台**](https://open.imdodo.com) 提供的API接口进行再封装，提供了目前开放平台提供的所有接口能力。

## 开源协议

​	本项目采用 [**MIT-Licenses**](https://choosealicense.com/licenses/mit/) 作为开源协议。

## 开发工具

​	PyCharm Professional Early Access Program
